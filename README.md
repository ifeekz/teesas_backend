# Teesa Backend

## Description

Built with [Nest](https://github.com/nestjs/nest), TypeORM and Posgres database.

## Installation

```bash
$ git clone https://github.com/ifeekz/teesas_backend.git
$ copy .env env.example
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
```
